ハラスメントとは？

定義

ハラスメント（Harassment）は、動詞Harassの名詞で「悩ますこと、いやがらせ、悩み(のたね)」という意味があります。日本では、相手の意に反する行為によって相手に不快な感情を抱かせる嫌がらせで、行為者がどう思っているかは関係なく、相手が不快な感情を抱けばハラスメントになります。

（※日本の人事部https://jinjibu.jp/keyword/detl/1035/）

実態

厚生労働省の公表によると、総合労働相談件数は11年連続で100万件を超え、ハラスメントは現代の日本社会でも大きな社会問題になっています。

その中でも、民事上の個別労働紛争相談件数における「いじめ・嫌がらせ」「自己都合退職」の相談件数は、10年前に比べて大きく増加しています。

一方で、「平成28年度　職場のパワーハラスメントに関する実態調査報告書」によると、企業調査の「パワーハラスメントの予防・解決のための取組の実施状況」と従業員調査の「勤務先のパワーハラスメントの予防・解決のための取組の実施状況」は、平成24年度に比べて増加していますが、「勤務先のパワーハラスメントの予防・解決のための取組の実施状況」の「ほとんど取り組んでいない」と「全く取り組んでいない」の回答比率は過半数を超えています。

また、「仕事の世界におけるハラスメントに関する実態調査2019」では、職場でハラスメントを受けたことがある人の半数近くが「誰にも相談しなかった」と回答し、その回答の理由のうち過半数以上が「相談しても無駄だと思ったから」という調査結果も出ています。

では、海外のハラスメント事情や世界から見た日本はどうなのでしょうか。

日本で初めてハラスメントに関する法律が規定されたのは1997年ですが、ヨーロッパでは1990年初期からハラスメントに関する法律が規定され始めました。

最新の「世界ジェンダー・ギャップ報告書（Global Gender Gap Report）2020」では、日本は121位となっており、G7の中で最下位となっています。

そして、2017年には#MeToo運動が世界中に広がり、2019年には国際労働機関（ILO）で「職場での暴力やハラスメントを全面的に禁止する」初の国際条約が採択されました。

ハラスメントに関する法制度面の取り組みは欧州諸国が進んでいるが、その他の国ではまだまだ進んでいるとは言えず、ここ数年でようやく世界的にハラスメントの取り組みが進められるようになってきたと考えられます。

（※Global Gender Gap Report 2020 | 世界経済フォーラムhttps://jp.weforum.org/reports/gender-gap-2020-report-100-years-pay-equality）

（※ハラスメント禁止条約、初の国際ルール、ILO総会　（写真=AP）　:日本経済新聞https://www.nikkei.com/article/DGXMZO46361440Q9A620C1EA1000/）

2020年6月には日本で「パワハラ防止法」が施行され、日本でもこれからさらにハラスメントに関する取り組みが進められるでしょう。
